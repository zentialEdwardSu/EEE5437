# EEE5437 Digital image compression

This repo holds the code, recommended reading materials, and written assignments for SUSTech EEE5437 Image and Video Compression and Network Communication.

code are built with a very simple, CMake based build system, most of it is written and implemented by AI. 

All the homework code can be built using `build.py`， so more detail，just have a look on `build.py --help`.
